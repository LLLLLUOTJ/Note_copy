摘要：
细胞内活性氧物质（ROS）的积累与线粒体功能和细胞凋亡紧密相连。ROS，包括超氧阴离子和过氧化氢等，既是细胞信号传导的参与者，又可能在过度积累时导致氧化应激，影响细胞生存。线粒体作为ROS的主要生成地，在氧化应激条件下，膜通透性增加，释放细胞凋亡相关蛋白，触发细胞凋亡。细胞凋亡是高度有序的细胞死亡，ROS通过调控凋亡信号通路和直接损伤细胞结构，直接影响凋亡的进行。这种细胞内ROS、线粒体和细胞凋亡的相互关系构成了一个复杂的调控网络，对于揭示细胞生物学机制和疾病治疗具有重要意义。

# 关键词
细胞凋亡，线粒体，ROS

# 引言 
细胞凋亡是一种高度调控的细胞死亡形式，涉及多种分子和信号通路。近年来的研究表明，金属离子、尤其是异价重金属离子，以及其他环境因子，如氟化物等，通过诱导细胞内活性氧物质（ROS）的积累，参与了细胞凋亡的复杂调控网络。金属离子的异常浓度和异价重金属离子的存在可能导致氧化应激，激活细胞内 ROS 的生成，从而影响线粒体功能、DNA 完整性以及激活 caspase 等凋亡相关蛋白。这些变化在细胞水平上逐步引导细胞朝向凋亡的方向演进。

内在细胞凋亡途径是线粒体引发的事件，涉及多种非受体介导的刺激，包括氟化物、自由基和缺氧[@jhaFluorideEnvironmentIts2011; @yinHypoxiainducedApoptosisMouse2018]。这些刺激针对线粒体内膜，导致线粒体通透性转换孔打开、MMP 丢失、ATP 合成减少以及 Bax 易位到线粒体中，从而导致 Cyt c 从线粒体释放到细胞质中，一旦释放到细胞质中，Cyt c 就可以与其他分子结合，形成凋亡小体，激活 caspase-3 和 caspase-9，引发细胞凋亡。Caspase是细胞凋亡过程中的关键执行者，其活性的调控与ROS密切相关。金属离子和异价重金属离子的存在可能通过直接或间接影响caspase的激活，介导凋亡信号的传递。氟化物等环境因子也被发现与ROS的生成和细胞凋亡的调控相关联。这些发现为我们深入理解细胞凋亡的分子机制提供了新的视角，同时也强调了环境因子对细胞命运的潜在影响。

在本文中，我们将探讨金属离子、异价重金属离子以及其他环境因子如氟化物等如何通过 ROS 的诱导影响细胞凋亡，深入解析这些因子在细胞生物学和疾病发展中的作用机制。这有助于为相关疾病的治疗和环境毒性的评估提供更为全面的认识。

# 异价重金属离子
在细胞面临不同价态的重金属 ( $Ag^+、Tl^+、Hg^{2+}、Cd^{2+}、Pb^{2+}、Al^{3+}、Ga^{3+}等$）的毒性作用下，线粒体和细胞将经历氧化应激过程，引发细胞凋亡，并在随后的病理生理过程中表现出复杂的特征。这一过程涉及到多个层面的细胞生物学和分子机制。[@korotkovMitochondrialOxidativeStress2023]

## 银
线粒体在呼吸链中有两个主要状态，分别是状态 3 和状态 4：状态 3（Phosphorylating State）： 也称为磷酸化状态，状态 3 是指线粒体在存在 ADP（腺苷酸二磷酸）和无机磷酸（Pi）的情况下进行活跃的 ATP 合成。在这个状态下，线粒体呼吸链通过负载电子的通路来产生 ATP。状态 3 通常与细胞对能量的高需求有关，因为 ADP 和 Pi 是 ATP 合成所需的关键成分。状态 4（Non-Phosphorylating State）： 也称为非磷酸化状态，状态 4 是指线粒体在没有足够 ADP 和 Pi 的情况下，ATP 合成相对较少，而线粒体主要通过呼吸链来维持基础的代谢活动。在状态 4 下，线粒体内膜通透性增加，质子通过内膜通道的速率增加，但不导致 ATP 的合成。状态 4 通常对应于细胞对能量需求较低的情况。这两种状态的相互转变反映了细胞对能量的实际需求，使线粒体能够适应不同的能量代谢状态。

低 $Ag^+$ 显示状态 4 增加，但高 Ag 导致 $\Delta \psi_{mito}$ 下降，完全抑制状态 3 和状态 4 呼吸，以及实验中内膜质子和钾渗透性、ROS 产生、脂质过氧化和肿胀的增加由于抑制线粒体 CI 和 CII 而不是 CIV，RLM 由谷氨酸或琥珀酸提供能量。只有 DTT（一种硫醇试剂）完全抑制这些 Ag 效应，而 MPTP 抑制剂（CsA 和 ADP）、乙二胺四乙酸（EDTA，一种二价金属螯合剂）和 RR（一种线粒体 Ca 2 单向转运蛋白阻断剂）则没有效果。在琥珀酸驱动的 RLM 中，S-15176 可抑制 Ag 诱导的肿胀、细胞色素 c 释放和 MPTP 打开，但 CsA 不会抑制。银纳米粒子 (AgNPs) 诱导 MPTP 打开，增加线粒体肿胀、4 态呼吸、脂质过氧化和 ROS 产生，以及 ΔΨmito 下降、3 态和 Ca 2 保留减少，以及在琥珀酸实验中可忽略不计的 3 UFCCP 下降。充满活力的 RLM。 AgNP 引起的肿胀的增加被 DTT 和乙二醇-双（β-氨基乙基醚）N, N, N', N'-四乙酸 (EGTA) 显着抑制，但不受 MPTP 抑制剂（ADP 和 CsA）的抑制，并且在存在 RR 的情况下，肿胀还会加速。 AgNP 诱导的线粒体肿胀在 $Ca^{2+}$ 负载的 RLM 中另外增加，随后 CsA 受到抑制。[@korotkovMitochondrialOxidativeStress2023]

重金属（$Ag^+、Tl^+、Hg^{2+}、Cd^{2+}、Pb^{2+}、Al^{3+}、Ga^{3+}等$）诱导线粒体通透性转换孔（MPTP）打开，同时细胞色素 c 释放到膜间隙中。这一过程的结果是，caspases-3 和-9 被激活并引发细胞凋亡诱导。一些金属（$Cd^{2+}、As^{3+}、Al^{3+} 和 In^{3+}$）会损伤 DNA，导致 bax 和 p53 激活，从而诱导细胞凋亡过程。箭头显示 MPTP 诱导。粗体垂直箭头显示重金属诱导细胞内凋亡过程中与线粒体相关的事件序列。与 DNA 损伤相关的细胞凋亡诱导途径由水平箭头表示。缩写：Cyt。 C、细胞色素 C； MPTP，线粒体通透性转换孔。[@korotkovMitochondrialOxidativeStress2023]

## 铊：
线粒体内膜对 $Tl^+$ 离子具有明显的渗透性，同时 $Tl^+$ 增加了 IMM （线粒体内膜）对单价阳离子（$K^+、H^+ 和 Na^+$）的被动渗透性， $Tl^+$ 通过线粒体 ATP 敏感钾通道 (mitoKATP) 和线粒体 BK 型 $Ca^{2+}$ 激活钾通道 (mitoKCa) 转运到通电的线粒体中，并使 MPTP（Mitochondrial Permeability Transition Pore，线粒体通透性转换孔）开启，MPTP 的开启可以导致线粒体内膜通透性的急剧增加，这会导致线粒体内外质子和离子的流失，以及其他重要的分子通透性的改变。这种状态的改变可能会引起线粒体肿胀、释放细胞色素 c 等事件，最终导致细胞凋亡。[@korotkovTlInducesBoth2013]

## 其它异价重金属离子：
$Ca^{2+}$ 和 ROS 作为生理学上的次要信使控制着决定细胞功能和钙离子依赖性和氧化应激反应的分子过程，能够激活细胞死亡效应器（半胱氨酸蛋白酶、神经酰胺酰胺、钙蛋白酶和 p 38），对线粒体和内质网造成不可逆的损害，并调节金属硫蛋白和 Bcl-2 蛋白的生物合成。

$Pb^{2+}$ 诱导细胞凋亡，包括激活 caspase -3和凋亡诱导因子的表达（Bax、Bcl2、Bcl-2）、MPTP 开放以及氧化应激。[@hanMitochondrionSensitiveTarget2021] $Cd^{2+}$ 会导致内质网和线粒体中储存的 $Ca^{2+}$ 的释放而引起细胞凋亡。[@biagioliEndoplasmicReticulumStress2008; @yuanCadmiumInducedApoptosisPrimary2013] $MeHg$ 与 $Hg^{2+}$ 类似诱导线粒体凋亡和氧化应激，介导 ROS 产生增加、细胞抗氧化防御能力下降以及线粒体释放细胞色素 c。$Hg^{2+}$ 诱导细胞凋亡，并激活 caspase-3、细胞活力下降、线粒体完整性下降和细胞内 ATP 耗竭。存在氧化应激，表现为 $\Delta \psi$ 线粒体下降、细胞色素 c 释放、ROS 生成增加、脂质过氧化增加、ATP 减少还原型谷胱甘肽。[@novoCellularMolecularMechanisms2021; @palanirajanHeavyMetalsMediatedPhospholipidsScrambling2020] $Al^{3+}$ 诱导的细胞凋亡伴随着 caspases-9、-8和-3的激活以及氧化应激[@xuFasMitochondriaMediatedSignaling2018]

## 总结：
分析了暴露于不同价态重金属（ $Ag^+、Tl^+、Hg^{2+}、Cd^{2+}、Pb^{2+}、Al^{3+}、Ga^{3+}等$）毒性作用的线粒体和细胞中发生的凋亡和氧化应激的原因和后果。所研究的重金属诱导了凋亡，表现为 caspase-3和-9的激活、Bax 和 Bcl-2的表达以及丝裂原激活蛋白激酶（ERK、JNK、p53 和 p38）的表达。观察到细胞活力降低和氧化应激，表现为 MPTP 开放、线粒体肿胀、ROS 和 $H_{2}O_{2}$ 产生增加、脂质过氧化、细胞色素 c 释放以及还原型谷胱甘肽和氧耗减少，以及由于来自内质网的 $Ca^{2+}$ 释放引起的细胞质和基质钙超载。还观察到由于 CI–CIII 功能障碍导致的 ATP 合成减少和 $\Delta \psi_{mito}$ 下降，这在体外细胞和线粒体实验中都有发现。这种功能障碍是由于一些金属（$Ag^6、Hg^{2+}、Cd^{2+}、Pb^{2+}、As^3+ 和 Sb^{3+}$）与呼吸复合物和腺苷酸转运蛋白的高亲和性巯基的相互作用引起的。另一方面，其他金属（$Al^{3+}、Ga^{3+}、In^{3+}、Cr^{6+} 和 U^{6+}$）诱导 $H_{2}O_{2}$ 和 ROS 的产生以及脂质过氧化，导致这些呼吸复合物的氧化损伤。一些金属（$Pb^{2+}、Al^{3+} 和 Ga^{3+}$）干扰 $Fe^{2+}$ 代谢并扭曲线粒体呼吸链铁硫中心的结构。同时，值得注意的是，在所研究的金属中，$Tl^+$ 具有显著的差异。一方面，在细胞实验中，$Tl^+$ 的毒性效应与其他重金属显示了明显的相似性。另一方面，在与分离的线粒体实验中，$Tl^+$ 并未诱导 MPTP 开放，与线粒体巯基的反应微不足道，并且不抑制呼吸酶的活性。$Tl^+$ 的毒性效应仅在钙负荷的线粒体实验中与其他重金属相似。因此，$Tl^+$ 的效应与其他重金属的相似性可能是由这些金属引起的胞质钙浓度增加引起的。

# 氟化物

氟以氟化物的形式广泛存在于环境中，是一种重要的微量元素，天然存在于土壤、水、多种疾病以及日常生活中使用的许多产品中。适量的氟化物对人和动物的生长具有有益作用，但人类和动物体内积累过量的氟，会对肾脏、肝脏、心脏、生殖系统、神经系统、肠道系统、内分泌系统等多个器官和系统产生不良影响[@jhaFluorideEnvironmentIts2011]。

ROS 被认为是细胞凋亡和线粒体自噬的驱动力[@cuiROSmediatedMitophagyApoptosis2021]，因为它们会诱导线粒体膜的通透性，损害呼吸链，最终触发细胞凋亡和线粒体自噬信号通路。大量 ROS 的诱导与线粒体依赖性细胞凋亡和 PINK1/Parkin 通路的激活直接相关，对成骨细胞产生明显的毒性，导致骨代谢异常。[@zhangFluorideExposureProvokes2023]

NaF 暴露增加了线粒体自噬相关蛋白（如 LC 3 II、PINK 1 和 Parkin）的水平，同时降低了 p 62 的表达，p 62 与 LC 3 II 相互作用，并在自噬溶酶体中被降解。此外，NaF 促进自噬通量的升高和酸性液泡的积累，这意味着自噬体与溶酶体融合形成自噬溶酶体。过量的氟化物会诱导线粒体自噬。酸性液泡通过自噬流将其内容物释放到骨细胞的微环境中，并且它可能诱导骨细胞周围骨的基质蛋白水解和脱矿质，随着时间的推移可能导致局部骨破坏。[@zhangFluorideExposureProvokes2023]

NaF 暴露增加了细胞内 ROS 和 mtROS 的产生和积累，并提高了 MDA 水平，同时降低了 SOD 活性和 GSH 含量，从而促进了氧化应激反应。这种氧化损伤反过来又会刺激 ROS 和 mtROS 的产生。与此同时，氟化物暴露引发了活性氧的产生并增加了氧化应激，与氟化物对骨骼的毒性，NaF 直接促进细胞内 ROS 和 mtROS 的产生，引发 G2/M 期阻滞，最终导致 MLO-Y4细胞中线粒体介导的细胞凋亡和线粒体自噬。这些细胞毒性作用被 ROS 清除剂 NAC 逆转，这意味着 NaF 暴露通过增加 ROS 产生诱导细胞凋亡和线粒体自噬。氟化物会减少细胞内 ATP 合成并触发 c-Jun N 末端激酶 (JNK) 介导的细胞凋亡。[@zhangFluorideExposureProvokes2023]

# 总结：

本文综合考察了线粒体与细胞凋亡之间的关系，特别关注了细胞内活性氧物质（ROS）在这一过程中的调控作用。通过深入分析异价重金属离子和氟化物对线粒体功能和细胞凋亡的影响，揭示了它们在调控细胞凋亡过程中的多样性机制。不同重金属离子引发的氧化应激过程涉及MPTP的开启、线粒体肿胀、ROS和H2O2产生的增加，最终激活了细胞凋亡途径，包括caspase-3和caspase-9的激活。这些变化由于金属离子与呼吸复合物和腺苷酸转运蛋白的相互作用引起，对ATP合成和ΔΨmito的下降产生影响。氟化物作为环境微量元素，其过量暴露导致了细胞凋亡和线粒体自噬的激活，通过ROS的增加和氧化应激的升高。这些细胞毒性效应通过ROS清除剂NAC逆转，进一步证明了氟化物通过增加ROS产生来诱导细胞凋亡和线粒体自噬。

这些研究为我们深刻理解生物体内 ROS 调控机制提供了新的视角，对于毒理学和环境科学具有重要意义。ROS 的产生和调控是生命体对外界环境刺激的敏感反应，也是环境因子影响生物体健康的关键因素。

在癌症治疗方面，调控ROS水平为一些新型治疗策略提供了可能性。除了癌症，这些研究还有助于理解其他疾病的发生机制，为疾病的早期诊断和治疗提供新的思路。因此，深入了解这些调控网络有望为推动环境健康、毒理学研究和疾病治疗领域的发展提供更为全面的认识。通过更深入地研究ROS的多层次调控机制，我们有望找到更加精准、高效的治疗策略，为维护人类健康和环境可持续发展贡献更多可能性。
